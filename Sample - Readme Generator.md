# Readme Generator 


  ## Description: 

  This application will automatically generate a readme.md file for a project of your choice by utilizing Node.js and following the prompts via Inquirer.

  ## Installation: 

  This application requires in the installation of Inquirer through 'npm install' in the command line.

  ## Usage: 

  A user will use 'node index.js' from the command line and follow the step-by-step walk through of the application.
